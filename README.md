# 1. 프로젝트 과정

💡 본 프로젝트는 기업과 연계한 산학 프로젝트입니다. 보안 및 이해상충 방지의 사유로 일부 수치와 이미지는 비공개 혹은 마스킹 처리되었습니다.

### 피드백 및 고찰
* 유저 행동 분석부터 시각화, 개선안 제안, A/B Testing까지의 과정이 수월하게 이뤄졌는가?

  → 분석 결과를 대시보드로 살펴보고, 이 과정에서 발견한 문제점을 해결해나가며 사용자 관점에서 유의미하고 지속 가능한 성과를 도출함.

* SQL을 활용하여 로그데이터를 효과적으로 추출했는가?

  → MongoDB 접근 권한을 받아 원하는 기간·기능에 맞는 로그데이터를 추출하였고, 이를 유저 행동 분석에 효과적으로 활용함.

* 기업과 연계된 산학 프로젝트를 진행하고 소통하는 과정에서 어려움은 없었는가?

  → 스타트업과의 기업 연계 프로젝트를 제안하고, 여러 커뮤니테이션 상황에서 기획·디자인·개발·마케팅 등 다양한 직군과의 협업 능력이 향상됨.
